# Student-Performance-Dashboard
The Student Performance Dashboard is a Power BI-based project that analyzes student academic data such as marks, attendance, and grades. It helps identify top-performing students, subject-wise performance, and overall academic trends.

The dashboard highlights key metrics like average marks, pass percentage, and attendance, allowing teachers and administrators to monitor student progress effectively. Interactive visualizations make it easy to explore patterns and insights.

Overall, this project demonstrates how data visualization can improve decision-making in education. It helps institutions enhance student performance and can be further improved with predictive analytics and real-time data integration.
